# Core Java Practice

This repository contains Java console-based programs created to practice
core Java and Object-Oriented Programming (OOP) concepts.

## Concepts Covered
- Classes and Objects
- Encapsulation
- Inheritance (basic)
- Polymorphism (basic)
- Conditional Statements
- Loops
- User Input Handling
- Java Random Class

## Projects Included
- Library Management System
- Stone Paper Scissors Game
- Guess the Number Game

## Tech Stack
- Java
- IntelliJ IDEA

## How to Run
1. Clone the repository
2. Open the project in IntelliJ IDEA (or any Java IDE)
3. Compile and run the main class

## Purpose
This repository was created for learning and strengthening Java fundamentals
through hands-on practice.
